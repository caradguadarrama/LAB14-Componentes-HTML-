# LAB14-Componentes-HTML-
Diseñar la pantalla de bienvenida de una aplicación inventada. No tiene que funcionar, solo debe verse bien.

Descripción:
Desarrollé esta interfaz de bienvenida para una aplicación ficticia. El ejercicio se centra en dominar las bases de maquetación web y diseño visual.

Inspiración Visual:
El diseño está inspirado en la estética del álbum "Brat" y el tráiler de "The Moment" de Charli XCX. Busqué evocar lo minimalista y disruptivo mediante:
El uso del color verde característico (#8ace00).
Tipografía minimalista con kerning (espaciado) ajustado y efectos de desenfoque (blur).
Un estilo crudo y moderno.

Modelo de Caja: Estructuración de contenedores mediante div y manejo de padding/margin.
CSS Moderno: Implementación de CSS Grid para el centrado absoluto y Variables CSS para un código más limpio.
Responsividad: Uso de la función clamp() para lograr tipografía fluida que se adapta a cualquier pantalla sin necesidad de tantos media queries.
Interacción: Micro-transiciones en el botón de "Empezar" mediante pseudoclases (:hover).
